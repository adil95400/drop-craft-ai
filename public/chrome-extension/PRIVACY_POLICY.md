# Privacy Policy - ShopOpti+ Chrome Extension

**Last Updated:** January 2025

## Overview

ShopOpti+ ("the Extension") is a Chrome browser extension designed to help e-commerce entrepreneurs import products, reviews, and monitor prices from various online marketplaces.

## Data Collection

### Data We Collect

1. **Product Information**
   - Product titles, descriptions, prices, and images from web pages you visit
   - Only collected when you explicitly use the import feature

2. **User Preferences**
   - Extension settings and configurations
   - Stored locally on your device

3. **Extension Authentication**
   - Secure tokens to connect your extension to your ShopOpti+ account
   - Stored locally and transmitted securely

### Data We DO NOT Collect

- Personal browsing history
- Passwords or payment information
- Personal identification information without consent
- Data from non-e-commerce websites

## How We Use Data

- **Product Import:** To help you import products to your ShopOpti+ dashboard
- **Price Monitoring:** To alert you of price changes on monitored products
- **Synchronization:** To sync your imported products with your account

## Data Storage

- All user settings are stored locally using Chrome's storage API
- Product data is transmitted securely to ShopOpti+ servers
- We use industry-standard encryption (HTTPS/TLS)

## Third-Party Services

The extension connects to:
- **ShopOpti+ API:** Our secure backend services
- **E-commerce platforms:** Only when you initiate product scraping

## Your Rights

You can:
- View all stored data through extension settings
- Export your settings at any time
- Clear all local data by uninstalling the extension
- Request deletion of your account data

## Permissions Explained

| Permission | Purpose |
|------------|---------|
| `activeTab` | Access current page for product scraping |
| `storage` | Save your preferences locally |
| `tabs` | Detect e-commerce websites |
| `scripting` | Inject import buttons on product pages |
| `notifications` | Alert you about price changes |
| `alarms` | Schedule automatic price checks |
| `contextMenus` | Right-click import options |
| `host_permissions` | Access e-commerce websites for scraping |

## Data Security

- All data transmissions use HTTPS encryption
- Authentication tokens are stored securely
- No sensitive data is logged or cached

## Children's Privacy

This extension is not intended for users under 13 years of age.

## Changes to Privacy Policy

We may update this policy and will notify users through the extension.
